Why We Chose COSMIC

In building LupineOS, we looked to the future. That future is being forged by System76 with the COSMIC Desktop Environment.

The Engineering First Approach

Drawing from System76's philosophy, COSMIC is designed to be the ultimate productivity environment. It treats the desktop as a workspace to be mastered, not just a launcher.

Built in Rust

COSMIC is the first mainstream DE written entire in Rust. This provides:

  • Memory Safety: Eliminating entire classes of bugs found in C/C++ desktops.
  • Performance: Highly optimized compositor for fluid 60+ FPS animations.

Tiling by Default

The auto-tiling feature creates an efficient flow, allowing you to navigate your windows with keyboard shortcuts. It's the perfect interface for the LupineOS "Pack" architecture, letting you manage multiple container terminals side-by-side effortlessy.
