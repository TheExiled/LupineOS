LupineOS v1.0: "WolfPack" Released

The wait is over. The first stable release of LupineOS is here, bringing the power of Atomic Fedora and the speed of COSMIC to your workstation.

What is LupineOS?

LupineOS is an immutable, atomic Linux distribution. This means the core operating system is read-only and updated in the background. If an update breaks something, you can simply reboot into the previous version.

Key Features in v1.0

  • Atomic Core: Built on Fedora Silverblue/Atomic technology.
  • COSMIC Desktop: A pure Rust-based desktop environment (Pre-Alpha build) that is fast and memory safe.
  • Gaming Ready: Now includes Wine, Lutris, Protontricks, and GameMode.
  • Privacy Focused: Dedicated "Privacy Vault" container with Tor and Gocryptfs.

The Pack System

We believe in keeping your host system clean. That's why we use Distrobox to manage development environments.

  • Alpha: Fedora-based container for general development.
  • Beta: Ubuntu-based container for sysadmin tools.
  • Omega: Kali Linux container for security testing.
  • Privacy Vault: Debian-based container for anonymous browsing and encryption.
